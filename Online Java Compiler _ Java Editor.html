Instructions to follow:
Library should be available in public Maven repository.
You can search maven artifacts in or .
It should be in a format like - org.apache.commons:commons-lang3:3.4.
On Maven, select "Gradle (short)" and copy the snippet. it will look something like this: implementation 'org.apache.commons:commons-lang3:3.4'.
Remove "implementation" and the single quotes and you are left with the correct format: org.apache.commons:commons-lang3:3.4.
Add your library to the field above, and don't forget to import it in your code too.
